A JavaScript Term Rewriting System

In a lost project Brian Beckman build a Mathematica style TRS and evaluation system in the browser. We are going to revive the ideas and present rwr the world's cutest JavaScript Term Rewriting System.

Build your own Vimium

Is there a minimal UI for the web we can extract from Vimium? Scroll, jump, and input focus are the killer features, VIM style text manipulation is out of scope because of the complexity there, thought that would be an interesting separate project.

Programming in Martin Lof's Type Theory

An interactive adaption of a classic book on programming using a direct translation of Martin Lof's full dependent type theory. Features a natural deduction interface.
